# ORM - Backend e-commerce

Description

A mysql database application which is backend for e-commerce site. by building this i used MySQL, dontenv, and Sequlize.

Installation 

npm i
npm install mysql
npm install sequlize


To Run
npm start
npm run seed 
control C 
mysql -u root -p
